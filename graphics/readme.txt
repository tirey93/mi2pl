CZCIONKI
Poniżej spis czcionek jakich używałem aby osiągnąć efekt widoczny w grze:

MENU GŁÓWNE, NORMALNY NAPIS
Comic Sans MS Bold Italic, 20px, efe7de, a potem Filters -> Light and Shadow -> Drop Shadow z ustawieniami: 
# GIMP 'Drop Shadow' settings

(time 0)
(x -0.25800000000000001)
(y -0.25800000000000001)
(radius 0)
(grow-shape square)
(grow-radius 1)
(color (color-rgba 0.35294117647058826 0.18823529411764706 0.031372549019607843 1))
(opacity 2)

# end of 'Drop Shadow' settings

MENU GŁÓWNE, ŚWIECĄCY NAPIS
Comic Sans MS Bold Italic, 20px, 5a3008, a potem Filters -> Light and Shadow -> Drop Shadow z ustawieniami: 
# GIMP 'Drop Shadow' settings

(time 0)
(x -0.25800000000000001)
(y -0.25800000000000001)
(radius 0)
(grow-shape square)
(grow-radius 2)
(color (color-rgba 0.90588235294117647 0.69803921568627447 0.38823529411764707 1))
(opacity 2)

# end of 'Drop Shadow' settings

MENU GŁÓWNE, POŚWIATA
Comic Sans MS Bold Italic, 20px, ffffff, a potem Filters -> Light and Shadow -> Long Shadow z ustawieniami: 
# GIMP 'Long Shadow' settings

(time 0)
(style finite)
(angle -31.559999999999999)
(length 10.6)
(midpoint 100)
(midpoint-rel 0.5)
(color (color-rgba 1 1 1 1))
(composition shadow-only)

# end of 'Long Shadow' settings
